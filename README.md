## Описание

Информационная система для учёта успеваемости студентов. Реализует четыре прецедента из диаграммы вариантов использования:

1. **Просмотр успеваемости** — фильтрация оценок по студенту, дисциплине, семестру, типу контроля
2. **Выставление оценки** — преподаватель выбирает студента/дисциплину, вводит оценку и комментарий
3. **Редактирование оценки** — поиск по ID и изменение существующей оценки
4. **Формирование отчёта** — сводный отчёт по группе и семестру с экспортом в TXT

## Стек

- Python 3.12 + Tkinter (GUI, входит в стандартную библиотеку)
- JSON (хранение данных)
- pytest + pytest-cov (тестирование, покрытие ≥90%)

```
├── models.py           # Бизнес-логика: User (ABC), Student, Teacher, Grade, Discipline, Group, Search
├── repository.py       # GradeRepository: CRUD + JSON-персистентность
├── ui/
│   ├── app.py          # Главное окно, ttk.Notebook
│   ├── view_tab.py     # Вкладка: Просмотр успеваемости

│   ├── edit_tab.py     # Вкладка: Редактирование оценки
│   └── report_tab.py   # Вкладка: Формирование отчёта
├── tests/
│   ├── test_models.py      # Тесты модельных классов
│   └── test_repository.py  # Тесты репозитория
├── data/
│   └── data.json       # JSON-хранилище (начальные данные)
├── main.py             # Точка входа
└── README.md
```
